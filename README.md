##Ejercicio1

### 1A. Pregunta
El texto del h1 no se centra visualmente porque su contenedor principal .site-header está usando display: flex con justify-content: space-between.

### 1B. Ejercicio - Soluciona de dos formas diferentes
Con Flexbox: Hacemos que el h1 ocupe el espacio central y usamos margin-left: auto en el menú para empujarlo a la derecha, asi el h1 queda en el centro del header.

Con Grid: Creamos como un contenedor con tres columnas: izquierda, centro, derecha.
Ponemos el h1 en la columna del centro y el menú en la derecha, para que el título quede perfectamente centrado.
